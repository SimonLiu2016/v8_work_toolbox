# 应用快捷键获取工具使用说明

## 功能介绍
应用快捷键获取工具是一个用于提取 macOS 应用程序快捷键信息的工具。它可以获取指定应用的所有菜单快捷键，并将其导出为格式化的文本文件。

## 文件格式
导出的文件为 txt 格式，包含三列信息，以 `|` 分隔：
- 描述：快捷键对应的功能描述
- 快捷键：具体的按键组合（如 Command+C）
- 分类：应用名称

## 权限设置
由于该工具需要访问 macOS 的辅助功能 API 来获取其他应用的菜单信息，因此需要以下权限设置：

### 1. 辅助功能权限
首次运行应用时，系统会弹出权限请求对话框，允许应用访问辅助功能 API。

如果未收到自动提示，您需要手动启用：
1. 打开系统设置（System Settings）
2. 进入隐私与安全性（Privacy & Security）
3. 选择辅助功能（Accessibility）
4. 点击下方锁图标解锁
5. 将 V8WorkToolbox 应用添加到允许列表中

### 2. 应用程序文件夹访问权限
为了能够发现和访问其他应用，您还需要授予应用访问应用程序文件夹的权限。

## 如何使用
1. 在主界面点击"应用快捷键获取"工具卡片
2. 输入目标应用的名称（如 "Finder", "Safari", "Visual Studio Code" 等）
3. 选择快捷键文件的保存位置
4. 点击"开始提取"按钮
5. 等待处理完成，结果将保存为 txt 文件

## 注意事项
- 目标应用必须正在运行，否则可能无法获取其快捷键信息
- 某些应用可能出于安全原因限制了快捷键信息的访问
- 请确保已正确授予所有必要权限，否则工具可能无法正常工作
- 该工具仅供个人使用，不建议用于商业用途

## 技术实现
- 使用 macOS 的 Accessibility API 获取应用的菜单信息
- 通过 Flutter 平台通道调用原生代码
- 支持递归遍历菜单结构以获取所有可用的快捷键

## 重要说明
由于 macOS 系统安全机制的限制，现代应用（包括系统应用和第三方应用）普遍限制了外部访问其菜单结构的能力。即使拥有辅助功能权限，许多应用也会阻止 Accessibility API 访问其菜单信息。这是正当的安全保护措施，旨在防止恶意软件窃取应用的菜单和快捷键信息。因此，此工具可能无法获取大部分现代应用的快捷键信息。

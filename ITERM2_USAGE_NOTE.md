# iTerm2 快捷键获取说明

如果您在使用应用快捷键获取工具时遇到无法获取iTerm2快捷键的问题，请按以下步骤操作：

## 1. 确认应用名称
- 尝试使用 "iTerm" 而不是 "iTerm2"
- 也可以尝试 "iTerm2" 

## 2. 确保iTerm2处于活动状态
- 在尝试提取快捷键之前，请确保iTerm2应用是当前活动窗口
- 点击iTerm2窗口以确保它获得焦点

## 3. 检查系统权限
- 打开「系统设置」→「隐私与安全性」→「辅助功能」
- 确保 V8WorkToolbox 应用已被勾选授权

## 4. 安全性考虑
请注意，由于iTerm2具有较高的安全级别，某些版本可能会限制对菜单和快捷键信息的访问，这是正常的安全保护机制。

## 5. 替代方案
如果仍然无法获取iTerm2的快捷键，您可以：
- 查看iTerm2的官方文档
- 在iTerm2的Preferences → Keys → Key Bindings中查看快捷键设置
